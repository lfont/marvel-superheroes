marvel-superheroes
==================
A list of Marvel Super Heroes.

Hacking
-------
1. Install dependencies  
`nvm use 6.9.4`  
`npm install`  
2. Does it work?  
`npm test`  
3. Build the bundle  
`npm run build`  
4. Let's Go!  
`env API_PRIVATE_KEY=[your private api key] npm start`  
5. Enjoy :)
